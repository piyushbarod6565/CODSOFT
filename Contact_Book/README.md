# 📒 Contact Book

A modern Contact Book application built using **Python** and **CustomTkinter**. It provides a clean and user-friendly interface for managing contacts with features like adding, updating, deleting, and searching contacts.

## ✨ Features

* ➕ Add new contacts
* ✏️ Edit existing contacts
* 🗑️ Delete contacts
* 🔍 Search contacts by name or phone number
* 💾 Store data in a local `contact.json` file
* 🌙 Light/Dark theme toggle
* 📱 Modern and responsive CustomTkinter UI
* ✅ Input validation for phone numbers and email addresses

## 🛠️ Technologies Used

* Python 3
* CustomTkinter
* Tkinter
* JSON
